Browser এর Size ছোট বড় করলে আমরা Effect টা দেখতে পাব। এখানে max Function যেভাবে কাজ করবে তা হলঃ এখান থেকে Maximum Value টি বের করবে। ধরি Browser এর Size 800px. এখানে 800px এর 50% হচ্ছে 400px. এখানে max function সবোর্চ্চ Value(400px > 300px) টাকে width হিসেবে নিবে। অর্থাৎ 400px হবে div এর width. এভাবে Browser এর Size ছোট বড় করলে ও Max Function এর ভিতরে সর্বোচ্চ Value কে width হিসেবে গ্রহণ করবে। তবে div এর width 300px এর নিচে হবে না।
